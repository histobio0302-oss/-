약가 안내 — 심평원 약가기준정보 API 연동
약품 상세의 📈 약가 안내 탭에서 이제 심평원 약가기준정보(약가목록조회)를 실시간으로 조회합니다. 현재 상한가 + 적용시작일자를 모든 등재 품목에 대해 표시하고, 제품코드 변경 이력(chgBfMdsCd 체인)을 자동으로 따라가 가능한 만큼의 과거 가격 변동 점을 차트에 함께 표시합니다. (기존 2008~2012 G-code 스냅샷은 보조 데이터로 함께 표시)

DUR 다중점검기 — 약품 여러 개 + 환자 정보로 한 번에 검사
의약품정보 → DUR 점검기 서브탭에서 최대 10개 약품을 입력해 병용금기·임부금기·노인주의·특정연령대금기·효능군중복 5종을 동시에 점검합니다. 입력 약품 간 페어 충돌(병용금기/효능군중복)을 별도로 강조해 표시하고, 나이·임신·65세+ 컨텍스트를 입력하면 해당 카테고리만 강조됩니다.

제형 분류 오류 수정 — 브랜드명 중간 '주'가 주사제로 잘못 분류되던 문제
동일성분 필터의 '주사제/정제' 등 제형 분류가 브랜드명 중간의 '주' 글자(예: 아주스타정·아주리정)를 주사제로 잘못 분류하던 문제를 수정했습니다. 이제 제품명 끝의 형태명(정·캡슐·시럽·주사제·연고·점안액·좌제·질정 등)만으로 정확히 분류하며, 함량·포장 표기(80/5밀리그램 등)는 무시합니다.
🎯
동일성분 — 심평원 주성분코드 기준으로 정밀화
동일성분 의약품을 심평원 주성분코드(약학정보원과 동일 기준)로 묶습니다. 성분뿐 아니라 함량·제형·배합비까지 같은 제품만 정확히 표시합니다. 예: 아목클라네오시럽(14:1) → 14:1 제품만, 7:1 제외.
🧬
동일성분 매칭 — 성분 색인 기반으로 전면 개선
제품명에 성분이 적혀 있지 않은 브랜드명 제품(예: 듀얼살탄·하이퍼스타)도 성분이 같으면 정확히 묶입니다. 식약처 주성분 데이터 4만여 제품을 성분세트로 색인화해, 표기(파우더·염) 차이와 무관하게 동일성분을 찾습니다.
📢
수정된 사항 — 첫 접속 시 팝업 안내
메인 페이지에 처음 접속하면 수정된 사항 팝업이 자동으로 한 번 표시됩니다. 한 화면에 3건씩 페이지로 넘겨볼 수 있고, 팝업 우상단 '가이드에서 보기'를 누르면 이 섹션으로 이동합니다.

의약품 검색
식약처·심평원 공공데이터 기반 의약품 단건 검색 및 다양한 부가 기능
01 기본 검색 SEARCH
제품명 또는 성분명으로 의약품을 검색합니다. 검색창 클릭 시 최근 검색어가 표시되어 빠르게 재검색할 수 있습니다.
▲ 검색 기준(제품명/성분명) 선택 후 검색어 입력 → 검색 버튼 클릭 또는 Ctrl+K

▲ 검색창 클릭 시 최근 검색어 목록 표시

▲ 카드형 검색 결과 — 제형, 허가일, 보험상한가 확인

💡 검색 팁: 제품명의 일부만 입력해도 검색됩니다. 예) "타이레놀", "아모잘탄", "플루코나졸"
성분명 검색은 공통 성분을 가진 의약품을 한번에 조회할 때 유용합니다.
02 검색 결과 보기 VIEW
검색 결과는 카드형과 리스트형 두 가지 보기 방식을 지원합니다. 화면 우측 상단의 아이콘으로 전환하며, 엑셀 내보내기도 가능합니다.
⊞
카드형 보기
각 의약품의 제형, 허가일, 보험상한가, 주성분 정보를 카드 형태로 표시합니다. 여러 항목을 한눈에 비교하기 좋습니다.

리스트형 보기
좁은 화면에서도 많은 결과를 한 번에 볼 수 있는 가로 행 형태입니다. 빠른 스크롤 탐색에 적합합니다.

▲ 카드형 보기

▲ 리스트형 보기

📥 엑셀 내보내기: 검색 결과 우측 상단의 엑셀 내보내기 버튼을 클릭하면 현재 검색 결과를 .xlsx 파일로 다운로드할 수 있습니다.
03 상세보기 DETAIL
카드의 상세보기 버튼을 클릭하면 해당 의약품의 3가지 상세 탭을 확인할 수 있습니다.
▲ 상세보기 — 의약품 상세정보 탭 (제품명, 허가업체, 포장규격, 생동성, 주성분 등)
  • 📋
    의약품 상세정보
    제품명, 허가업체(판매사), 제조원, 포장규격, 허가(변경)일, 제조 구분, 생동성 여부, 성상, 주성분 정보를 확인합니다.
  • 🧬
    동일성분 의약품
    동일한 주성분을 가진 의약품 목록을 조회합니다. 가격순·이름순·제조사순 정렬이 가능합니다.
  • 📈
    약가 안내
    심평원 약제급여목록 데이터 기반으로 최저가/현재가/최고가와 변동 이력을 차트로 시각화합니다.

▲ 동일성분 의약품 — 가격순 정렬

▲ 약가 안내 — 이력 차트 및 통계

04 제네릭 조회 GENERIC
카드의 제네릭 버튼을 클릭하면 해당 의약품과 동일 성분을 가진 제네릭 의약품 전체 목록을 가격순으로 조회합니다.
▲ 제네릭 목록 — 총 192개, 가격순/이름순/제조사순 정렬 지원
🥇 순위 메달: 가격순 정렬 시 상위 3개 제품에 금/은/동 메달이 표시됩니다.
05 의약품 비교 COMPARE
두 의약품을 나란히 비교할 수 있습니다. 카드의 비교 버튼으로 A/B 슬롯에 담은 후 비교 모달을 열면 주요 항목이 나란히 표시됩니다. 동일한 값은 회색, 차이가 있는 항목은 색상으로 구분됩니다.
▲ 의약품 비교 — 허가업체, 포장규격, 생동성, 보험상한가, 주성분 등 항목별 비교
1
A 의약품 선택
첫 번째 의약품 카드의 비교 버튼 클릭 → A 슬롯에 등록
2
B 의약품 선택
두 번째 의약품 카드의 비교 버튼 클릭 → B 슬롯에 등록
3
비교 확인
화면 하단의 비교 플로팅 바에서 비교하기 클릭 → 항목별 비교 모달 확인
06 내 목록 추가 MY LIST
카드의 + 목록추가 버튼으로 관심 의약품을 목록에 담을 수 있습니다. 목록에서 DUR 병용금기 일괄 확인 및 엑셀 다운로드가 가능합니다.
▲ 내 목록 — 추가된 의약품 목록, DUR 병용금기 확인, 엑셀 다운로드
⚠️ DUR 병용금기 확인: 목록에 담긴 의약품들 간의 병용금기 여부를 일괄 확인합니다. 처방 전 안전성 검토에 활용하세요.
07 링크 공유 SHARE
카드의 공유 버튼을 클릭하면 해당 의약품의 식약처 의약품안전나라 상세 페이지 링크를 공유하거나 QR코드를 확인할 수 있습니다.
▲ 링크 공유 — URL 복사 또는 QR코드로 공유

안전정보
식약처 의약품 회수·판매중지 및 행정처분 정보 조회
01 회수·판매중지 조회 RECALL
식약처에서 고시한 의약품 회수·판매중지 목록을 조회합니다. 제품명으로 검색하거나 전체 조회로 최신 목록을 확인하세요.
▲ 회수·판매중지 목록 — 제품명, 업체명, 회수사유, 회수명령일자, 구분(자진/강제) 표시
▲ 상세정보 — 회수사유, 품목기준코드, 의약품안전나라 바로가기 링크
🚨 주의: 강제회수 배지가 표시된 항목은 식약처의 강제 명령에 의한 회수입니다. 즉시 사용 중단을 권장합니다.
02 행정처분 조회 ADMIN
식약처의 의약품 행정처분 이력을 조회합니다. 제품명 또는 업체명으로 검색하거나 최근 처분 버튼으로 최신 목록을 확인합니다.
▲ 행정처분 조회 — 제품명, 업소명, 행정처분명, 처분일자, 공개종료일
▲ 행정처분 상세정보

의료기관 검색 (병원·약국)
심평원 HIRA 건강지도 공식 서비스 연동
01 HIRA 건강지도 연동 MAP
심평원 공식 건강지도 서비스를 포털 내에서 바로 이용할 수 있습니다. 지역·종류별 병원·약국 검색, 우리 지역 좋은 병원 찾기, 전문병원 찾기 등의 기능을 제공합니다.
▲ HIRA 건강지도 — 병원/약국 위치 지도 검색, 종류별 필터, 다운로드 지원
  • 🗺️
    병원·약국 지도 검색
    시/도 → 시/군/구 → 읍/면/동 단계로 지역을 선택하거나 병원명·약국명을 직접 검색합니다.
  • 🏷️
    종류별 찾기
    병원·약국을 종류별(상급종합, 종합병원, 의원, 치과, 한방 등)로 필터링합니다.
  • ⭐
    우리 지역 좋은 병원
    심평원 의료기관 평가 결과를 기반으로 우수 의료기관을 조회합니다.
  • 🔬
    전문병원 찾기
    복지부 지정 전문병원 목록을 확인합니다.
↗ 새 탭에서 열기: 우측 상단의 새 탭에서 열기 버튼을 클릭하면 심평원 건강지도 공식 사이트를 전체 화면으로 이용할 수 있습니다.
02 병의원 개원정보 NEW
심평원 병원정보서비스의 개설일자를 기준으로 그 달에 신규 개원한 병의원을 종별로 보여줍니다.
▲ 상단: 커스텀 월 선택기(년도 segmented + 1~12월 chip) + 시도·시군구·종별 필터
  • 📅
    월 선택 + 지역·종별 필터
    조회월·시도(필수)·시군구·종별(다중) 조합으로 결과를 좁힙니다.
  • 🔍
    키워드·정렬·뷰 전환
    이름/주소 즉시 필터, 4종 정렬, 📑 그룹 ↔ 📋 리스트 토글, 📥 엑셀 내보내기.
  • 📋
    상세 모달 — 의료기관 상세 + 우수기관평가
    카드의 📋 상세 클릭 시 진료시간·응급실·주차·진료과목별 전문의수와 ⭐ 우수기관평가 등급이 한 모달에 표시됩니다.

▲ 시도(필수) → 시군구 → 종별 chip 다중 선택

▲ 요약 바: 전체 카운트 + 종별 분포 chip

▲ 📋 리스트 뷰: 종별 색상 상단 띠 + 동일 크기 상세/지도/홈 버튼 (의원 파랑, 한의원 초록 등)
💡 활용: 신규 개원 동향·경쟁 분석·우수 평가기관 발굴 등. 심평원 공공 API 실시간 조회 (회원가입 불필요).

의약품 정보
허가정보, DUR 안전정보, 보험인정기준, 약제기준목록표 조회
01 허가정보 조회 LICENSE
식약처에 허가된 의약품의 공식 허가정보를 조회합니다. 제품명 또는 업체명으로 검색할 수 있으며, 품목기준코드 링크로 식약처 상세 페이지를 확인합니다.
▲ 의약품 허가정보 — 제품명, 업체명, 제형, 허가(변경)일, 품목기준코드
02 DUR 안전정보 조회 DUR
심평원 DUR(의약품 적정사용 서비스) 안전정보를 조회합니다. 병용금기, 임부금기, 노인주의, 특정연령대금기, 효능군중복 등 5가지 항목을 한번에 확인할 수 있습니다.
▲ DUR 안전정보 — 병용금기·임부금기·노인주의·특정연령대금기·효능군중복 항목별 조회
항목 설명
⚠️ 병용금기함께 복용하면 안 되는 다른 성분 목록
← 임부금기임산부에게 복용이 금지되거나 주의가 필요한 경우
👴 노인주의고령 환자에게 주의가 필요한 경우
👶 특정연령대금기특정 연령대(소아, 고령자 등)에서 사용이 제한되는 경우
💊 효능군중복동일 효능군에 속하는 성분 목록
검색 유형: 전체 항목 외에 병용금기, 임부금기, 노인주의 등 특정 항목만 선택해 조회할 수 있습니다.
03 보험인정기준 COVERAGE
심평원 보험인정기준 고시·행정해석·심사지침 등을 조회합니다. 약제 급여 기준 관련 최신 고시 정보를 확인할 수 있습니다.
▲ 보험인정기준 — 고시, 행정해석, 심사지침, 심의사례공개, 자료실 등 탭별 조회
↗ 새 탭에서 열기: 심평원 보험인정기준 공식 사이트를 새 탭으로 열어 전체 기능을 이용할 수 있습니다.
04 약제기준목록표 FORMULARY
심평원 약제기준정보 목록표를 조회합니다. 급여 약품의 분류·성분·함량·보험상한가 등 기준 정보를 확인합니다.
▲ 약제기준정보 목록표 — 심평원 급여 약제 기준 정보
💊
비급여진료비
건강보험심사평가원 공공데이터 기반 자기부담 진료비 비교
01 병원명 검색 HOSPITAL
병원명, 지역, 종별로 비급여진료비 항목을 검색합니다. 해당 병원에서 청구하는 비급여 진료항목과 금액을 확인할 수 있습니다.
▲ 병원명 검색 결과 — 비급여항목명, 병원표기명, 현재금액, 적용기간, 상세 바로가기
🏷️
검색 조건
병원명(일부 입력 가능), 지역(시/도), 종별(상급종합/종합병원/병원/의원 등) 조합으로 검색합니다.
↗
상세 바로가기
검색 결과의 바로가기 링크를 클릭하면 심평원 비급여 공개 사이트에서 상세 정보를 확인합니다.
02 지역별 비교 REGION
특정 비급여 항목의 지역별·종별 가격을 비교합니다. 항목코드를 입력하면 해당 지역의 병원들이 청구하는 비용 범위를 확인할 수 있습니다.
▲ 지역별 비교 — 항목코드 기준 지역 내 최소/최대 비용 비교
1
항목코드 확인
항목코드 조회 탭에서 원하는 항목(예: 초음파, 1인실)을 검색하여 코드를 복사합니다.
2
지역·종별·항목코드 입력
지역별 비교 탭에서 지역, 종별, 항목코드를 입력합니다.
3
결과 확인
해당 지역·종별 병원의 최소·최대 비용 범위와 병원별 상세 금액을 확인합니다.
03 항목코드 조회 CODE
비급여 진료 항목의 표준 코드를 검색합니다. 검색어를 입력하거나 빈칸으로 두면 전체 항목 코드 목록을 조회합니다.
▲ 항목코드 조회 — 코드 클릭 시 자동 복사, 지역별 비교 탭에 붙여넣기 사용
💡 코드 복사: 코드 칸을 클릭하면 클립보드에 자동 복사됩니다. 지역별 비교 탭에 붙여넣어 바로 사용하세요.

자주 조회하는 의약품을 저장하고 관리
01 즐겨찾기 관리 FAVORITES
자주 조회하는 의약품을 즐겨찾기에 저장해두면 빠르게 재조회할 수 있습니다. 즐겨찾기된 약품은 탭 버튼에 숫자 배지로 표시됩니다.
▲ 즐겨찾기 — 저장된 의약품 카드, 상세보기 및 링크 공유 지원
  • ⭐
    즐겨찾기 추가·삭제
    의약품 검색 결과 카드의 즐겨찾기(★) 버튼으로 추가하고, 즐겨찾기 탭에서 × 버튼으로 삭제합니다.
  • 📋
    상세보기
    즐겨찾기 카드의 상세보기 버튼으로 의약품 상세정보, 동일성분 의약품, 약가안내을 바로 확인합니다.
  • 🔗
    링크 공유
    즐겨찾기 카드의 링크 아이콘으로 해당 의약품의 식약처 상세 페이지를 공유합니다.
  • 💾
    브라우저 저장
    즐겨찾기 데이터는 브라우저 로컬 스토리지에 저장되어 별도 로그인 없이 유지됩니다. (브라우저 캐시 삭제 시 초기화)
📱 모바일 접속: 모바일에서 접속 시 화면 하단의 네비게이션 바에서 즐겨찾기 탭을 바로 선택할 수 있습니다. 저장된 개수가 배지로 표시됩니다.
