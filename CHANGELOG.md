# 📝 변경 이력 (CHANGELOG)

## v1.4.0 - 제형 필터링 추가 (2026-05-13)

### ✨ 주요 기능
**제형별 동일성분 검색 - 약학정보원과 동일한 방식**

이제 "신델라주(주사제)" 검색 시:
- ✅ 티옥트산 **주사제**만 11건 표시
- ❌ 티옥트산 정제, 캡슐 등은 제외

### 🔧 개선사항
- **제형 자동 추출** (30+ 제형 지원)
  - 주사제, 정제, 캡슐제, 시럽제
  - 서방정, 필름코팅정, 장용정
  - 연고, 크림, 로션, 패취
  - 좌제, 점안액, 흡입제 등
- 콘솔에 추출된 제형 표시
- 성분 + 제형 이중 필터링

### 📊 예시
```
"신델라주(주사제)" 검색
→ 성분: 티옥트산
→ 제형: 주사제
→ 결과: 티옥트산 주사제만 11건 ✅
```
