# 🧪 동일성분 검색 테스트 가이드

## 🎯 기대 결과

### 예시 1: 에이미카시방정 (미라베그론)
- **검색어**: "에이미카"
- **추출된 성분**: 미라베그론
- **기대 결과**: 23건 이상의 동일성분 제품

### 예시 2: 타이레놀 (아세트아미노펜)
- **검색어**: "타이레놀"
- **추출된 성분**: 아세트아미노펜
- **기대 결과**: 다수의 동일성분 제품

## 🔍 테스트 방법

### 1. 브라우저 개발자 도구 열기
```
Chrome: F12 또는 Ctrl+Shift+I
Edge: F12
Firefox: F12
```

### 2. 콘솔 탭으로 이동
- Console 탭 클릭
- 로그 메시지를 확인할 수 있는 화면

### 3. 의약품 검색
```
1. "의약품 검색" 탭 클릭
2. 검색창에 "에이미카" 입력
3. "검색" 버튼 클릭
```

### 4. 상세보기 클릭
```
1. 검색 결과에서 첫 번째 제품 찾기
2. "📄 상세보기" 버튼 클릭
3. 모달 창이 열림
```

### 5. 동일성분 탭 클릭
```
1. "🧬 동일성분 의약품" 탭 클릭
2. 자동으로 검색 시작
3. 결과 확인
```

### 6. 콘솔 로그 확인
정상 작동 시 다음과 같은 로그가 표시됩니다:
```
원본 주성분: 미라베그론-50밀리그램-서방성필름코팅정(미라베그론)
추출된 성분: 미라베그론
전체 검색 결과: 156건
필터링 전: 156건
필터링 후: 23건
중복 제거 후: 23건
```

## ✅ 성공 기준

### 정상 작동
- ✅ 콘솔에 "전체 검색 결과: N건" 표시
- ✅ 콘솔에 "필터링 후: N건" 표시 (N > 0)
- ✅ 테이블에 제품 목록이 표시됨
- ✅ 제품명, 제조사, 포장규격이 올바르게 표시됨

### 실패 사례
- ❌ "동일성분 의약품을 찾을 수 없습니다"
  - 원인: API 검색 결과 없음
  - 해결: 성분명 확인 또는 다른 약품으로 테스트
  
- ❌ "동일성분 의약품이 없습니다"
  - 원인: 필터링 후 결과 없음
  - 해결: 콘솔에서 필터링 로직 확인

- ❌ "오류: [메시지]"
  - 원인: API 호출 실패 또는 네트워크 오류
  - 해결: 네트워크 탭에서 API 응답 확인

## 🐛 문제 해결

### Case 1: 결과가 0건
```javascript
// 콘솔에서 확인
원본 주성분: [확인]
추출된 성분: [확인]
전체 검색 결과: 0건  ← 문제!
```

**해결책:**
1. 성분명이 올바른지 확인
2. API 키가 유효한지 확인
3. 네트워크 탭에서 API 응답 확인

### Case 2: 필터링 후 0건
```javascript
// 콘솔에서 확인
전체 검색 결과: 100건
필터링 전: 100건
필터링 후: 0건  ← 문제!
```

**해결책:**
1. 제품명 패턴 확인
2. 성분명 매칭 로직 재검토
3. MAIN_ITEM_INGR 필드 확인

### Case 3: API 오류
```javascript
// 콘솔에서 확인
GET https://apis.data.go.kr/... 403 (Forbidden)
```

**해결책:**
1. API 키 만료 확인
2. 서비스 키 재발급
3. 네트워크 방화벽 확인
